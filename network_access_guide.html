üìã Troubleshooting Steps

1

Check Network Connection

Ensure both devices are on the same WiFi network or local network.

2

Find Your IP Address

If the IP changed, run this command to get the current IP:

ip addr show | grep "inet 192"
3

Test Local Access First

On this computer, test: http://192.168.0.104:9001

4

Check Mobile Device

On your mobile device or other computer, open a web browser and navigate to:

http://192.168.0.104:9001
5

Router/Network Issues

If still not working, check:

  • Router firewall settings
  • AP isolation (guest network restrictions)
  • Port forwarding (not usually needed for local network)
